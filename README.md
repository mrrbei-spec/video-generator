# video-generator
video-generator 技能是一个强大的AI视频生成工具，它的核心价值在于：通过Type×Style系统和Superpowers框架，将内容转化为具有专业视觉冲击力和情感共鸣的视频作品。
##核心能力
1. 8种视频类型 × 14种视觉风格
类型：explainer（讲解）、story（故事）、tutorial（教程）、promo（营销）、cinematic（电影级）、slideshow（幻灯片）、animated-infographic（动态信息图）、pptx-to-video（PPT转视频）
风格：craft-handmade（手工质感）、chalkboard（黑板教学）、minimal（极简）、bold-editorial（杂志风格）、cute（可爱）、warm（温暖）、technical-schematic（技术图纸）、pixel-art（像素风）、cyberpunk-neon（赛博朋克）、watercolor（水彩）、vintage（复古）、flat-vector（扁平矢量）、algorithmic（算法生成）
2. Superpowers框架
这是技能的灵魂，10种"超能力"让视频超越普通解释：
Pattern Hunter（模式猎手） ：揭示隐藏的模式和关联
Bridge Builder（桥梁构建者） ：连接看似无关的概念
Storyteller（讲故事的人） ：将枯燥事实转化为情感共鸣的叙事
Simplifier（简化者） ：将复杂概念拆解为直观隐喻
Amplifier（放大器） ：最大化关键信息的冲击力
Perspective Shifter（视角转换者） ：从全新角度呈现熟悉概念
Empathy Engine（共情引擎） ：通过人性元素建立情感连接
Pattern Breaker（模式打破者） ：刻意打破预期制造惊喜
Time Bender（时间操控者） ：通过时间变化创造戏剧性
Sensemaker（意义制造者） ：从混沌中提炼秩序
3. 完整的视频生产工作流
分镜规划 → AI媒体生成 → 文案创作 → TTS语音合成 → FFmpeg专业后期处理
支持多种AI媒体生成API（Stable Diffusion、DALL-E、Runway、Pika、Sora）
ChatTTS多语言TTS语音合成，支持发音变体和音频缓存
Graphiti时态知识图谱，提供个性化推荐和上下文管理
4. 设计系统集成
Remotion理念：声明式动画、帧级精确控制、数据驱动视觉
Canvas设计原则：三分法、黄金比例、网格系统、视觉层次
前端设计美学：独特字体、大胆配色、拒绝通用AI风格
小红书设计美学：新鲜、精致、年轻化、社交优化

触发方式
当你需要生成视频时，这个技能会自动激活。

具体场景包括：
显式触发
"生成一个视频" / "create a video"
"制作视频" / "视频生成"
任何明确要求视频产出的指令

隐式触发（根据内容自动选择）
当用户提供的内容需要通过视频形式表达时：
教程类：包含"步骤"、"如何"、"学会"等关键词
故事类：包含"经历"、"旅程"、"情感"等叙事元素
营销类：包含"发布"、"推广"、"产品"等商业关键词
讲解类：包含"解释"、"什么是"、"概念"等教育性内容
数据可视化：包含"数据"、"统计"、"图表"等信息

典型使用场景
教育内容：制作教程视频、知识讲解视频
营销推广：产品宣传片、品牌故事视频
数据呈现：动态信息图、数据可视化视频
社交媒体：抖音/小红书短视频、Instagram Reels
企业演示：PPT转视频、产品演示视频
创意表达：艺术短片、视觉诗歌

技术特色
中文人性化表达：避免翻译腔，使用自然、地道的中文表达，匹配受众文化背景
说服性文案：遵循AIDA/PAS模型，优化标题、钩子和行动号召
视觉技能应用：思维导图、流程图、概念图、时间线、组织架构图、知识图谱
FFmpeg专业处理：质量优化、视觉特效、硬件加速、批量处理
小红书风格优化：emoji丰富、数字列表、社区互动、真实可信
这个技能不仅仅是"生成视频"，它是将内容策略、视觉美学和情感工程融为一体的视频创作系统。它拒绝平庸的AI视频，追求每一帧都有设计意图，每一秒都在传递情感和信息价值。
